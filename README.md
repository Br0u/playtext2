# playtext2

`playtext2` is a single-page interactive text work built around `Attention Is All You Need`.
It mixes a playable text field with a more settled reading state, using real paper excerpts,
classical book-like styling, and width-aware text layout.

## References

- Interaction direction: `paper-pong`
- Text layout inspiration: `@chenglou/pretext`
- Source paper: [`Attention Is All You Need`](https://arxiv.org/abs/1706.03762)

## Local Development

```bash
npm install
npm run dev
```

## Commands

```bash
npm test
npm run build
```

## Notes On Content

The current dataset is adapted from the paper abstract and short excerpted passages for
interactive presentation. It does not present generated text as if it were original paper language.
